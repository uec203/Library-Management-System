# Library Management System

This is an offline Library Management System developed using Java Swing and JDBC.
The application helps manage books, members, and book issue/return operations.

## Technologies Used
- Java
- Java Swing
- JDBC
- MySQL / SQLite
- Eclipse

## Features
- Add, update, delete books
- Search books
- Manage library members
- Issue and return books
- Admin login
- Offline database system


## Database Tables
- books (book_id, title, author, quantity)
- members (member_id, name, contact)
- issue (issue_id, book_id, member_id, issue_date, return_date)

## How to Run
1. Open the project in your Eclipse
2. Create the database and import as maven project
3. Configure database details in DBConnection.java
4. Run App.java
